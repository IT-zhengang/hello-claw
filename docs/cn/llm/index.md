---

AI大模型架构解析这一部分，聚焦模型层面的能力边界与结构理解。

它会承接智能体之下、更偏模型系统的问题，例如：

- 上下文窗口与注意力分布
- 推理链路与思维模式
- 多模态模型的输入输出机制
- 模型能力边界与工程取舍

这一部分会和“构建龙虾”“AI Agent智能体”形成互补：

- 构建龙虾：讲系统底座
- AI Agent智能体：讲 Agent 能力设计
- AI大模型架构解析：讲模型本身的结构与能力边界

除了深度专题之外，这一部分也会补充更适合入门读者的图解导读页，先帮助你把模型、Agent、MCP、工作流与 Memory 这些概念放回同一张地图里，再进入后续章节。
