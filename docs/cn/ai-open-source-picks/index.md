AI 开源精选集收录的是那些值得开发者认真读源码、看设计、评估落地价值的 AI 开源项目。

它不是“新闻快讯”，也不是“仓库大全”，而是一份更偏工程判断的精选书单：只挑那些真正会影响 Agent 执行环境、工具调用、安全边界、浏览器自动化、远程开发和工作流搭建的项目。

和“构建龙虾”不同，这个目录不聚焦 OpenClaw 自身，而是把视野放到外围生态：哪些基础设施值得借鉴、接入、复用，甚至直接放进你自己的 AI 系统里。

## 这个目录会写什么

这里的每一篇文章，都会尽量回答四个问题：

- 这个项目到底解决了什么问题
- 它的架构设计有没有参考价值
- 它和 Agent / OpenClaw / Coding 工作流有什么关系
- 它是否值得你真的花时间部署、接入或二次开发

## 建议阅读顺序

1. 如果你正在做 AI 编码、代码执行、浏览器自动化或远程开发环境，先看 [OpenSandbox](./opensandbox/)
2. 如果你想理解 OpenClaw 自身的 Runtime、提示词、工具和消息循环，回到 [构建龙虾](/cn/build/)
3. 如果你更关心“落地成什么场景”，继续看 [龙虾大学](/cn/university/intro.html)
